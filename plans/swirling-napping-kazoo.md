# 登录和注册页面测试方案

## Context

TechCraft 项目的登录和注册功能已开发完成，采用验证码登录方式和4步向导式注册流程。为确保功能稳定性和用户体验，需要制定全面的测试方案。

**技术架构**：
- 前端：HTML5 + CSS3 + 原生 JavaScript
- 测试方式：手动功能测试 + 浏览器开发者工具辅助
- Mock数据：assets/data/auth-mock.js

---

- [register.html](register.html) - 注册页面

## 五、测试执行计划

### 5.1 测试环境

```
操作系统：Windows 11 / macOS 14 / Ubuntu 22.04
浏览器：Chrome 120+, Firefox 120+, Safari 17+, Edge 120+
网络：本地开发服务器 (http://localhost:8000 或 VS Code Live Server)
```

### 5.2 测试准备

1. **环境搭建**
   - 启动本地开发服务器
   - 打开浏览器开发者工具
   - 清除所有缓存和存储

2. **测试数据准备**
   - 客户账号：13800138000 / 123456
   - 开发者账号：13900139000 / 123456
   - 新注册手机号：18888888888

3. **测试工具**
   - 浏览器开发者工具（Console、Network、Application）
   - 响应式设计模式（F12 → Ctrl+Shift+M）

### 5.3 测试执行顺序

```
第1天：登录页面测试（L-001 ~ L-036）
第2天：注册页面测试（R-001 ~ R-037）
第3天：集成测试 + 兼容性测试（I-001 ~ I-006, C-001 ~ C-005）
```
